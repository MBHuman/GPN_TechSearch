Project for GPN Tech.Challenge
-----------
This is Search Engine for GazpromNeft(GPN)

Main advantages of SearchEngine:
* Fast
* Simple

How to run project:
* `source env/bin/activate` for use venv
* `python3 app.py` for run app

Used instruments:

* Yandex MapsAPI
* Selenium SearchRobot
* Flask
* Python
* Bulma
* Pandas
* NetworkX
* Gensim
